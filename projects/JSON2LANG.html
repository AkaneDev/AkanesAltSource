JSON2LANG - Minecraft Language File Backporting Tool

Repository: GitHub

Author: AkaneDev

Version: 0.1 alpha

Overview

JSON2LANG is a command-line utility designed to backport Minecraft translation files from modern JSON-based formats to the legacy .lang format used in Minecraft Forge 1.12.2. It automates the conversion process for modders who need compatibility with older versions.

Purpose

  • Minecraft 1.12.2 uses .lang files for localization.
  • Minecraft 1.13+ switched to JSON-based translation files.
  • JSON2LANG converts newer JSON translation files into the .lang format required for Minecraft 1.12.2.

How It Works

Input Format (JSON)

{
    "block.minecraft.stone": "Stone",
    "item.minecraft.diamond": "Diamond"
}

Output Format (.lang)

block.minecraft.stone=Stone
item.minecraft.diamond=Diamond

Usage

Run JSON2LANG with the following command:

jsontolang [-h] input output

Arguments

  • input - Path to the input JSON file.
  • output - Path where the .lang file will be saved.

Example

jsontolang en_us.json en_us.lang

Why Use JSON2LANG?

  • Fast and Efficient - Converts entire files in seconds.

Future Improvements

  • Batch conversion for multiple files.
